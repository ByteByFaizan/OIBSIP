# Weather Forecast Dashboard (Python GUI)

An advanced desktop-based weather application built using Python and Tkinter that provides real-time weather data, forecasts, charts, and alerts using the WeatherAPI service.

This project evolves the Basic Weather App idea into a feature-rich GUI dashboard, suitable for college projects, resumes, and GitHub portfolios.

---

## Project Overview

The Weather Forecast Dashboard allows users to:
- Search weather by city name
- View current weather conditions
- Analyze hourly and 5-day forecasts
- Switch between Celsius and Fahrenheit
- Visualize temperature trends using interactive charts
- See air quality index (AQI) and weather alerts

---

## Project Levels

### Beginner Concepts Covered
- API integration using requests
- JSON data parsing
- User input handling
- Error handling (timeouts, invalid cities)
- Command-line to GUI transition

### Advanced Features Implemented
- Tkinter-based modern GUI
- Multi-threading for non-blocking API calls
- Weather icons using Pillow (PIL)
- Embedded Matplotlib charts
- Unit conversion (°C ↔ °F)
- Hourly and multi-day forecasts
- Air Quality Index (AQI)
- Weather alerts
- Responsive UI with hover effects
