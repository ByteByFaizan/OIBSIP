# OIBSIP – Python Programming Projects

This repository contains Python projects developed during the **AICTE Oasis Infobyte Internship Program (OIBSIP)**.  
The projects focus on applying core Python concepts to build practical, beginner-to-intermediate level applications.

As per internship guidelines, **three projects** have been successfully completed and uploaded.

- **Program:** AICTE Oasis Infobyte Internship

---

## Projects Overview

### 1. Advanced Password Generator
A Python-based application designed to generate strong and secure passwords using randomization techniques.

**Features:**
- Custom password length selection
- Combination of uppercase, lowercase, digits, and special characters
- Focus on password strength and unpredictability

**Technologies Used:**
- **Language:** Python 3
- **GUI Library:** Tkinter
- **Security:** `secrets` module
- **Standard Libraries:** `string`
**Project Folder:** `Advanced_Password_Generator`

---

### 2. BMI Calculator
A graphical BMI (Body Mass Index) calculator that computes BMI using user-provided height and weight and classifies health status.

**Features:**
- GUI-based input system
- Accurate BMI calculation
- Health category classification
- Data storage and visualization

**Technologies Used:**

| Technology | Description |
|----------|------------|
| Python | Core programming language |
| Tkinter | GUI framework |
| SQLite | File-based persistent storage |
| Matplotlib | Data visualization |
| datetime | Timestamp management |


**Project Folder:** `BMI_Advanced_Project`

---

### 3. Weather Application
A Python weather application that fetches real-time weather data for a specified location using an external API.

**Features:**
- Live weather data retrieval
- Displays temperature, humidity, and conditions
- Responsive GUI with background threading

**Technologies Used:**
| Category | Tools |
|--------|------|
| Language | Python 3 |
| GUI | Tkinter |
| API | WeatherAPI.com |
| Charts | Matplotlib |
| Images | Pillow (PIL) |
| Networking | Requests |
| Threading | threading |


**Project Folder:** `weather_app`
